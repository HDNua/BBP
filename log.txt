RXPB v3.6.0.0 EnemyScript v2.0.0 (IMPORTANT)

* !!!!! IMPORTANT !!!!!
  The AnimationClip 'DashFogEffect' used by the Animation component 'EffectDashFog' must be marked as Legacy.
> Unity 5에서는 Animation Component 대신 Mecanim이라는 애니메이션 시스템을 도입했는데,
  이때문에 Animation 컴포넌트를 직접적으로 개체에 붙이면 이런 경고가 뜬다.
  따라서 Unity 개체 선택 화면에서 Animation 컴포넌트를 제거하거나, Debug 모드에서 Legacy 체크를 하는 방법이 있는데
  후자쪽이 여러모로 불편하므로 그냥 전자를 추천함.
  

- 차지샷 시작 시 발생하는 효과 개선.
> 이제 점프 시에 챠지 샷을 발사해도 효과가 더 좋게 나옵니다.


! 차지샷 스프라이트의 색깔을 보다 화려하게 바꾸면 더 좋지 않을까 합니다.


! 대쉬할 때 플레이어의 Collider2D가 변해야 한다.


- StageManager의 _playerX, _playerZ 필드를, public으로 선언하는 대신
  DataBase로부터 가져와서 초기화하는 방식으로 변경.
  
  
- 게임 타이틀 이미지(SnakeHead) 크기 변경.
  
  
- GameData 필드 업데이트.
> TryCount 속성 추가.


- HUDScript 정의 변경.
> 원래 Camera에 붙어있던 것을 HUD 개체로 옮김.
>> CameraFollow야 Camera 것이 맞는데, PauseUI는 또한 PauseUI 개체로 옮기는 것이 마땅하지 않나 한다.


* !!!!! IMPORTANT !!!!!
  MonoBehaviour 개체의 enabled를 조작해서 화면에 나타나게 하는 대신,
  gameObject.SetActive(true)를 사용해야 한다. 즉 이래야 함.
  
  _HUD.enabled = true; // wrong code
  _HUD.gameObject.SetActive(true); // good code
